# 3D Family Tree Visualization

An interactive 3D family tree visualization built with React, TypeScript, and react-force-graph-3d. Multiple family clusters are spatially separated in 3D space and connected through marriage links, with Google Earth-style navigation for immersive exploration.

## Features

- 3D force-directed graph visualization
- Multiple family clusters with spatial separation
- Marriage links bridging between clusters
- Google Earth-style navigation (orbit, zoom, pan, click-to-focus)
- Node metadata display (name, birth date, birth place)
- Different link types (parent, sibling, marriage) with visual distinctions

## Tech Stack

- React + TypeScript
- react-force-graph-3d (Three.js/WebGL)
- Vite
- Vercel (deployment)
